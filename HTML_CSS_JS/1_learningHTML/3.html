  Get
      URL 改变，在URL 里显示 HTML Form 参数的 name/value 值。
      只适合有少量参数的 HTML Form，因为 URL 长度有字符限制，不能无限长。
      涉及安全性的信息，比如用户密码，不能用 get，因为会在 URL 上显示，不安全。
  Post
      URL 不改变，不在 URL 里显示 HTML Form 的数据。
      Form 提交的信息没有长度限制。
      涉及安全性的信息，如用户密码，应采用 post 方式。
